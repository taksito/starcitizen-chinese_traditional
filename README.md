簡單操作步驟如下
1. 點選 code ，然選 download zip (下載zip)
2. 解壓縮zip，然後進去有一個 starcitizen-chinese_traditional的資料夾再點進去可以看到有user.cfg還有data的資料夾。
3. 解壓縮到 star citizen的 live資料夾即可。
