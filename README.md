🛠 安裝教學 (簡單三步驟)
點擊 GitHub 頁面上的 [Code] 按鈕，選擇 [Download ZIP]。

解壓縮下載的檔案，進入資料夾後會看到 user.cfg 與 data 資料夾。

將這兩個檔案/資料夾直接複製到星際公民安裝路徑下的 LIVE 資料夾內即可。

註：若想移除漢化，只需刪除 user.cfg 檔案，非常安全。
